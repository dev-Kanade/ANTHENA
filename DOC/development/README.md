# 開発ドキュメントとロードマップ
このOSS開発に着手する場合はディベロップメントドキュメントをよく読んでください。
また、`master`ブランチにはプッシュしないでください。

## 技術スタック
このソフトウェアの開発をおこなうには以下の技術スタックが必要です。

- Rust
- PostgreSQL(SQL)
- ネットワーク
- tokio.rs
- 他...今後詳しいスタックをドキュメント化します。

## ソフトウェアの概要
プロジェクトルートに設置してある[README](../../README.md)に書いてある通り本ソフトウェアは認証及び認可フローの代案であり、現在はLinux/Debian系のOSであり、APTパッケージをサポートしているOSのみサポートしています。</br>
今後対応予定のOSと対応方法はロードマップやそれに関連するディベロップメントドキュメントを読んでください。
